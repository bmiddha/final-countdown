## About this project

Final Exam Schedule for UIC with countdowns.

This is a progressive web app (PWA) written in [TypeScript](https://www.typescriptlang.org/) and [Next.js](https://nextjs.org/).

## Getting Started

- Dependencies: `yarn install`
- Developing: `yarn dev`
